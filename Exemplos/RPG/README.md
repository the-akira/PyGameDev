## Controls

### Movement

- `w`: Move up
- `a`: Move left
- `s`: Move down
- `d`: Move right

### Combat

- `i`: Cast fireball up
- `j`: Cast fireball left
- `k`: Cast fireball down
- `l`: Cast fireball right

### Menu

- `Esc`: Pause menu

## Maps

Maps can be loaded with [Tiled](https://www.mapeditor.org/).
